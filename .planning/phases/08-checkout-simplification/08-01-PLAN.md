<objective>
Simplify the order backend: remove payment method and shipping method requirements, make address fields optional, default to pickup, and simplify the order text generator.

Purpose: The store currently only supports pickup (no delivery) and has no payment processing. The 3 payment methods and 3 shipping methods are misleading UI theater. Strip them out at the backend level so the checkout can become a simple contact form.

Output: Simplified order validation, service, and text generator. DB migration making address/shipping/payment columns nullable.
</objective>

  zipCode: z.string().max(20).optional().nullable(),
